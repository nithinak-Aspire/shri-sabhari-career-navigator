# Master Prompt to Paste into Claude Code

You are a senior digital product designer, information designer, and front-end engineer.

Work inside this repository and improve the existing Shri Sabhari Career Navigator. Read `CLAUDE.md` before making changes.

## Goal
Create a premium, self-explanatory employee-facing career navigator that explains the four Shri Sabhari career levels and how growth happens. It must work as:
1. an interactive web experience,
2. a 5-page presentation,
3. a printable 16:9 PDF.

## Required content
- Slide 1: Your Career Journey at Shri Sabhari
- Slide 2: The four levels — CL4, CL3, CL2, CL1
- Slide 3: Vertical, specialist, and lateral growth
- Slide 4: Six readiness areas
- Slide 5: Five-step usage guide

## Design expectations
- Use Shri Sabhari as the primary identity and Aspire as the secondary partner.
- Respect all palette and hierarchy rules in `CLAUDE.md`.
- Improve the layout, motion, micro-interactions, visual clarity, and mobile responsiveness without making it decorative or playful.
- Do not add stock photography.
- Do not use a conventional corporate template.
- Use geometric pathways, subtle line art, white space, and strong typographic hierarchy.
- The four-level framework must be understandable in under 30 seconds.

## Workflow
1. Inspect all files and explain the current architecture.
2. Run a local server and review the page visually.
3. Make a concise improvement plan.
4. Implement the changes.
5. Test desktop, mobile, keyboard navigation, interactions, and print output.
6. Report what changed and any remaining issues.

Do not ask for confirmation unless a decision would materially change the approved brand hierarchy or career framework content. Proceed with reasonable design judgement.
